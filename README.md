# Cyclistic Bike-Share Analysis 

## Project Overview
This project analyzes how **annual members** and **casual riders** use Cyclistic bikes differently.
The goal is to provide **data-driven marketing recommendations** to convert casual riders into annual members.

This case study follows the **Google Data Analytics process**:
Ask → Prepare → Process → Analyze → Share → Act.

---

## Business Task
Identify key differences in bike usage between casual riders and annual members to support membership growth strategies.

---

## Data Source
- Public Divvy bike-share data (Chicago)
- Source: https://divvy-tripdata.s3.amazonaws.com/
- Time period analyzed: Previous 12 months

> Note: Data is anonymized and complies with privacy requirements.

---

- SQL / R / Excel / Google Sheets (choose what YOU used)

## Key Findings
- Casual riders take **longer rides on average**
- Members ride more frequently on **weekdays**
- Casual usage peaks on **weekends**, suggesting leisure behavior

## Recommendations
1. Promote **weekend membership trials** for casual riders
2. Offer **discounted annual plans** after repeat casual usage
3. Use **digital ads and email campaigns** targeting high-frequency casual riders
